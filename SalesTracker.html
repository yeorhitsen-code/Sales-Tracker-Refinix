📋 Paste from Google Sheets / Excel

Fastest way — no file saving needed:

  1. In Google Sheets, make sure your first row is the header (see column names below)
  2. Select all your data rows including the header
  3. Press Ctrl + C to copy
  4. Click inside the box below and press Ctrl + V to paste
  5. Click Import Pasted Data
